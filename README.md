# 🍽️ Zomato Data Analysis using Python

## Project Overview

This project analyzes Zomato restaurant data to understand customer preferences, restaurant ratings, online ordering trends, and cost patterns using Python.

## Dataset

- Zomato Restaurant Dataset

## Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights

- Most restaurants provide online ordering.
- Majority of restaurants have ratings between 3.5 and 4.0.
- Certain restaurant types receive more votes.
- Customer spending patterns vary across restaurants.
- Ratings and votes are positively related.
